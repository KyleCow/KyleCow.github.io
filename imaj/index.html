Imaginary Proof

`(a+bi)^(c+di)`

Given that `e^(ln(x))=x`
We can set `(a+bi)^(c+di)` as x
And since in logs we can take powers to the outside it becomes
`e^((c+di)*ln(a+bi))`
Now we need to find what `ln(a+bi)` is

`ln(a+bi)`

Since all complex numbers are points on a plane
They can be expressed in a polar form (angle and magnitude)
These values are shown in the equation
`me^(itheta)`
Which can also be expressed as
`e^(ln(m)+itheta)`
The magnitude can simply be calculated using the pythagorean theorum `a^2+b^2=c^2`
With the magnitude being the hypotenuse length it can be expressed as `sqrt(a^2+b^2)`
The angle can also be calculated by using the `tan` ratio
`tantheta=(opp)/(adj)`
and since `a` is adjacent and `b` is opposite
the angle is `tan^(-1)(b/a)`

Now we know the angle and the magnitude so we know that the point in polar form is
`e^(ln(sqrt(a^2+b^2))+itan^(-1)(b/a))`
so
`ln(a+bi)=ln(sqrt(a^2+b^2))+itan^(-1)(b/a)`
and `ln(sqrt(a^2+b^2))=ln((a^2+b^2)^(1/2))=1/2*ln(a^2+b^2)`
so `ln(a+bi)=1/2ln(a^2+b^2)+itan^(-1)(b/a)`

`e^((c+di)(1/2ln(a^2+b^2)+itan^(-1)(b/a)))`

If we expand the bracket keeping in mind that `i^2=-1`
`(c+di)(1/2ln(a^2+b^2)+itan^(-1)(b/a))`
becomes
`c/2ln(a^2+b^2)+d/2ln(a^2+b^2)i+ctan^-1(b/a)i-dtan^-1(b/a)`
Which can be expressed as
`(c/2ln(a^2+b^2)-dtan^-1(b/a))+i(d/2ln(a^2+b^2)+ctan^-1(b/a))`
since we need to raise e to the power of this we get
`e^((c/2ln(a^2+b^2)-dtan^-1(b/a))+i(d/2ln(a^2+b^2)+ctan^-1(b/a)))`
or
`e^(c/2ln(a^2+b^2)-dtan^-1(b/a))*e^(i(d/2ln(a^2+b^2)+ctan^-1(b/a)))`
To make things clearer we will set
`theta = d/2ln(a^2+b^2)+ctan^-1(b/a)`
`omega = c/2ln(a^2+b^2)-dtan^-1(b/a)`
so we now have `(a+bi)^(c+di)=e^omega*e^(itheta)`
Since according to Eulers identity `e^(itheta)=costheta+isintheta`
This can be written as `e^omega(costheta+isintheta)`
Expressed fully it's
`(a+bi)^(c+di)=e^(c/2ln(a^2+b^2)-dtan^-1(b/a))cos(d/2ln(a^2+b^2)+ctan^-1(b/a))+e^(c/2ln(a^2+b^2)-dtan^-1(b/a))sin(d/2ln(a^2+b^2)+ctan^-1(b/a))i`

or

`(a+bi)^(c+di)=e^omega(costheta+isin(theta))`
when
`theta = d/2ln(a^2+b^2)+ctan^-1(b/a)`
`omega = c/2ln(a^2+b^2)-dtan^-1(b/a)`
